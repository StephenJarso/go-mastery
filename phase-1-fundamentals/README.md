# Phase 1: Go Fundamentals

**Focus:** Variables, data types, control flow, functions, slices, maps, and pointers.

---

## 🎯 Learning Objectives

After completing this phase, you'll be able to:
- Declare variables using `var` and the short declaration syntax `:=`.
- Work with Go's basic types: integers, floats, booleans, and strings.
- Control execution flow using `if-else` conditionals, `for` loops, and `switch` statements.
- Write functions with multiple return values and named returns.
- Understand value semantics vs. pointer semantics in Go.
- Create and manipulate basic data collections: arrays, slices, and maps.
